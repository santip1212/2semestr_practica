Для запуска приложение требуется:
Установить Qt по ссылке: https://www.qt.io/download-qt-installer-oss (у меня получилось скачать только с VPN).
Перед установкой лучше заранее завести аккаунт. 

После скачивания открываем Qt. 

В правом верхнем углу(или через файл в правом верхнем углу) окрываем папку, в которой находится проект.
После ищем CMakeLists.txt и открываем его.
После чего запускаем код.
